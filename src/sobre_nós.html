Introdução

Nós buscamos oferecer uma porta de entrada para pessoas que estão pensando em iniciar carreira na área de desenvolvimento.

Dicas

FRONT END

Trabalha com a parte visual do site, focado em design.
Principais linguagens Front End - HTML,CSS e JavaScript

BACK END

Voltado para a lógica do site, onde todas as regras de negócio serão manipuladas.
Principais linguagens Back End - Python, PHP e Java.

DATABASES

Foca principalmente na manipulação de dados.
Principais sistemas de Database - SQL, MySQL e NoSQL

Sobre Nós

A Easy Code foi criada com o intuito de fornecer um aprendizado nas diversas linguagens de programação de forma rápida, interativa e didática.
